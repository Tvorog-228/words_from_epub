# 📚 EPUB Word Extractor

A high-speed utility script designed to extract unique vocabulary from `.epub` books.

### 🎯 The Goal
I needed an efficient way to pull words from specific books that I knew contained the perfect vocabulary for another project: a **Flashcard Generator** (word + definition).

### ⚡ Performance
This extractor is built to be lightweight and extremely fast, optimized to handle large volumes of text in record time:
* **Speed:** Capable of processing two massive books and generating the word set in **approximately 1 second**.
* **Precision:** Automatically filters out punctuation and numbers, converting everything to lowercase to eliminate duplicates.

### 🛠️ Requirements
* Python 3.12+
* `ebooklib` & `beautifulsoup4`
